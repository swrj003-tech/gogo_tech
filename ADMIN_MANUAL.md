# Admin Portal Manual

## Accessing the Admin Dashboard

1.  Navigate to `https://your-website.com/admin`.
2.  If not logged in, you will be redirected to the secure login page.
3.  Enter your **Email** and **Password** (provided by your system administrator).
4.  Click **Access CRM**.

## Managing Content

### Pages
-   **Edit Pages**: Modify content for Home, About, etc.
-   **Translations**: Manage English and French content side-by-side.
-   **SEO**: Update meta titles and descriptions for better search ranking.

### Blog
-   **Create Post**: Add new articles with rich text, images, and tags.
-   **Publishing**: Save as draft or publish immediately.

### Leads
-   View incoming quote requests from the website.
-   Export leads to CSV.

## Security Features

-   **Session Timeout**: You will be automatically logged out after 24 hours of inactivity.
-   **Secure Login**: Powered by AWS Cognito (Enterprise-grade security).
-   **Audit Logs**: All sensitive actions (publishing, deleting) are logged.
